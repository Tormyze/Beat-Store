# 🎵 Beat Store - Landing Page
Projeto de uma **Landing Page** moderna para showcase de uma **Beat Store** (Loja de Beats), focada em performance e design responsivo.

## 🚀 Sobre o Projeto
Landing page pessoal desenvolvida para produtores musicais e beatmakers com o intuito de expor e vender seus beats de forma profissional, um website totalmente atrativo em design e com alta qualidade responsiva.
Criado em visão de uma interface limpa, rápida e que funcione também em dispositivos móveis de forma fluida.

## 🧧 Idealização
Além de Dev, sou **Beatmaker** e decidi criar este projeto como forma de demonstrar minha visão na **construção de software**, **código eficiente** e **desenvolvimento web**, podendo divulgar minhas obras como produtor musical e aumentar meu portfólio como futuro Engenheiro de Software.

## 🛠️ Tecnologias
Linguagens e tecnologiaas utilizadas na construção do projeto:
* **HTML5** - Estruturação semântica do site, com containers flexíveis.
* **CSS3** - Estilização completa, Media Queries, responsividade.
* **JavaScript** - Funcionalidades interativas (principalmente no carrosel).
* **Git** - Versionamento e controle de commits.

## ✨ Features
- [x] Paleta de cores focada em contraste.
- [x] Seção de apresentação dos beats.
- [x] Design adaptável para dispositivos móveis.
